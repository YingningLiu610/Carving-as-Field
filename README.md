# Carving as Field

Author: Yingning Liu

Date: April 2026

Course: Computational Arts - WCC2 - Individual Project

## Description

Carving as Field is a 20-minute performative work that explores carving as a time-based and generative process.

## Instructions

1.Open Max/MSP

Open the Max patch in the project folder

Select the correct audio input/output (audio interface required)

Click Start Script to begin sending data

2.Start the server

In VS Code (project root), run:

npm install

node server.js

3.Open the visual system

In a browser, go to:

http://localhost:3001

4.Run the performance

Begin carving

The system will respond in real time through sound and visual generation

## Notes

This system is designed to run locally

Real-time audio input is required

A contact microphone and audio interface are necessary for proper interaction

## Acknowledgements

This project builds on experiments in generative systems, sound-based interaction, and networked communication.

Tools and frameworks used include p5. js, Max/MSP, and WebSocket-based data transmission.
